# Projekt 3 - E-shop

## Spustenie

Spustenie webovej aplikácie:
```console
foo@bar:~$ sudo docker-compose build
foo@bar:~$ sudo docker-compose up web
```

Spustenie testu:
```console
foo@bar:~$ sudo docker-compose up test
```
